npx cypress run : browser açmadan bütün tesleri calıştırır.
                  Hata alınan testlerin ekran görüntülerini screenshots klasörüne koyar
                  Bütün testlerin video kayıtlarını video klasörüne koyar.

npx cypress info: Bilgisayarda yüklü browser bilgilerini, cypress versiyonu,
                  işletim sistenini, hafıza bilgileri gibi verileri gösterir

npx cypress verify : cypress in yüklü oldugunu onaylar

npx cypress run --browser chrome: chrome açmadan tüm testler çalışır

npx cypress run --browser chrome --headed : chrome açarak tüm testleri çalıştırı

İSTENEN KLASÖRDEKİ TÜM TESTLERİ ÇALIŞTIRMAK 
npx cypress run --spec 'cypress/e2e/day03/'
*default olarak Electron ile çalıştırır

npx cypress run --spec 'cypress/e2e/day03/' --browser chrome
*day03 tüm testleri chrome headless browser açmadan ile çalışır

npx cypress run --spec 'cypress/e2e/day03/' --browser chrome --headed
*day03 tüm testler chrome browser açarak çalıştırır

TEK BİR TESTİ ÇALIŞTIRMAK İÇİN
npx cypress run --spec npx cypress run --spec 'cypress/e2e/day02/01_LoginTest1.cy.js'
**default olarak Electron ile çalıştırır

npx cypress run --spec npx cypress run --spec 'cypress/e2e/day02/01_LoginTest1.cy.js' --browser chrome
*day02 klasöründeki 01_LoginTest1.cy.js dosyasını chrome açmadan çalıştırır

npx cypress run --spec npx cypress run --spec 'cypress/e2e/day02/01_LoginTest1.cy.js' --browser chrome --headed
*day02 klasöründeki 01_LoginTest1.cy.js dosyasını chrome açarak çalıştırır
